# Security Policy

## Reporting Security Issues

Please do not open a public issue for a security vulnerability.

Report security concerns through GitHub's private vulnerability reporting if it is enabled for the repository, or contact the repository owner through a private channel.

## Data Safety

This project renders local HTML files and writes local output artifacts. Before using it on sensitive material:

- Review the HTML source and local assets.
- Avoid embedding secrets, tokens, private customer data, or production credentials in examples.
- Inspect generated PDFs and screenshots before sharing them externally.
