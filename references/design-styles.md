# Design Styles

Use these directions as starting points. Keep each deck coherent rather than mixing all styles.

## Executive Brief

- Audience: founders, senior operators, buyers, investors.
- Mood: calm, premium, decisive.
- Layout: wide margins, compact grids, clear claims, few decorative elements.
- Color: off-white, ink, teal or green accents, restrained neutrals.
- Components: key judgment blocks, scope tables, timeline strips, risk/status markers.

## Editorial Report

- Audience: clients reading a polished narrative report.
- Mood: magazine-like, spacious, high-touch.
- Layout: large title blocks, pull quotes, feature bands, strong page rhythm.
- Color: light paper background, deep ink, one accent color.
- Components: quote bands, insight cards, section dividers, image placeholders when relevant.

## Operations Dashboard

- Audience: teams comparing metrics, tasks, risks, and progress.
- Mood: dense, useful, controlled.
- Layout: dashboard grid, tables, compact status labels, repeated information modules.
- Color: light UI background, neutral panels, green/amber/red status accents.
- Components: KPI tiles, action lists, swimlanes, exception tables, progress bars.

## CJK Typography

Use font stacks that include common CJK fonts:

```css
font-family: "Inter", "Noto Sans CJK SC", "Noto Sans SC", "PingFang SC", "Microsoft YaHei", sans-serif;
```
